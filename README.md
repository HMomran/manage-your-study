# Manage Your Study

A simple study planner built with Next.js, TypeScript, and Tailwind CSS.

## What it does

- Add subjects and the number of lessons for each subject.
- Choose a start date and end date.
- View one study day at a time.
- Check off each subject task when it is finished.
- Carry unfinished material forward to the next day automatically.
- Reset the planner back to an empty state.

## Local Development

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000/study`.

## Build

```bash
cd frontend
npm run build
```
